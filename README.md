# Proyecto Flask - Escuadrón de Superhéroes

Aplicación web hecha con Flask para buscar, filtrar y ver detalles de superhéroes a partir de un archivo JSON.

## JSON utilizado
URL original del JSON: PON_AQUI_LA_URL

## Modificaciones realizadas
- Se añadieron más héroes al archivo JSON.
- Se repitieron habilidades entre varios héroes para que el filtro tuviera más sentido.
- Se mantuvo la estructura general del archivo.

## Funcionalidades
- Página principal con imagen
- Búsqueda por nombre
- Filtro por habilidad
- Orden alfabético ascendente y descendente
- Vista de detalle de cada héroe
